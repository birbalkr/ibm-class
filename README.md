# 🤖 Full-Stack AI Cloud Web App

This is a full-stack AI-powered web application built using **React.js (Vite)** for the frontend and **Node.js (Express)** for the backend. The project integrates IBM Cloud services like App ID, Cloudant, Watson Chatbot, and Text to Speech to create a smart and interactive user experience.
